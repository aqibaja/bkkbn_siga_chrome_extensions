📥 Step 1: Download Ekstensi

Dapatkan File Ekstensi

  1. Buka halaman web tutorial ini
  2. Klik tombol "Download Ekstensi"
  3. File SIGA_Smart_Downloader.zip akan terunduh
  4. Ekstrak file ZIP ke folder yang mudah diakses
  5. Anda akan mendapat folder berisi file ekstensi

� Tip: Simpan di folder seperti C:\Extensions\SIGA_Downloader agar mudah ditemukan

🔧 Step 2: Instalasi Ekstensi

Pasang di Browser Chrome/Edge

  1. Buka browser Chrome atau Edge
  2. Ketik chrome://extensions/ di address bar
  3. Aktifkan Developer Mode (toggle di kanan atas)
  4. Klik tombol "Load unpacked"
  5. Pilih folder hasil ekstrak ekstensi
  6. ✅ Ekstensi akan muncul di daftar & siap digunakan

⚠️ Penting: Pastikan Developer Mode aktif, jika tidak tombol Load unpacked tidak muncul

🖥️ Step 3: Mengenal Interface

Tampilan Popup Ekstensi

  • 🏢 Header: Logo BKKBN & judul ekstensi
  • 📑 3 Tab Utama: TAHUNAN, BULANAN, DOWNLOAD
  • 🔗 Form URL: Textarea untuk input URL (bisa banyak, pisah baris)
  • ☑️ Checkbox Kabupaten: Pilih 1 atau lebih dari 23 kab/kota
  • 🔘 Filter Tambahan: Periode, Tahun, Kecamatan, Desa (sesuai kebutuhan)
  • 📊 Tab Download: Monitor status & progress real-time

2 Cara Sesuai Spesifikasi Laptop

Multi URL - Single Kabupaten
✓ Untuk laptop kuat (RAM ≥8GB)
✓ Buka banyak tab sekaligus

Single URL - Multi Kabupaten
✓ Untuk laptop standar

✓ Lebih stabil (serial)

🚀 METODE 1: Multi URL - Single Kabupaten

Untuk Laptop Kuat (RAM ≥8GB)

� Kapan Menggunakan Metode Ini?

  • ✓ Laptop Anda memiliki RAM 8GB atau lebih
  • ✓ Ingin download banyak laporan berbeda sekaligus
  • ✓ Koneksi internet stabil & cepat
  • ✓ Mengutamakan kecepatan (paralel processing)

📊 Contoh Skenario:

Download 23 jenis tabel untuk 1 kabupaten = Buka 23 tab sekaligus

🚀 Metode 1: Langkah-Langkah

Multi URL - Single Kabupaten

  1. 📋 Salin SEMUA URL laporan yang ingin didownload (5-20 URL)
  2. Paste URL ke textarea (pisahkan dengan enter/baris baru)
  3. ☑️ Pilih HANYA 1 KABUPATEN saja
  4. Pilih Periode & Tahun (jika Tab Bulanan)
  5. Klik Submit
  6. 🌐 Ekstensi akan membuka banyak tab sekaligus (1 tab per URL)
  7. � Monitor progress di Tab Download
  8. 💾 File otomatis tersimpan di folder Downloads

🚀 Metode 1: Contoh Praktis

Download 3 Laporan Bulanan - Aceh Besar

� Input Textarea (3 URL):

https://siga.bkkbn.go.id/tabel1
https://siga.bkkbn.go.id/tabel2
https://siga.bkkbn.go.id/tabel3

☑️ Pilih: HANYA "Aceh Besar"

� Periode: Januari | Tahun: 2025

✅ Hasil: 3 tab terbuka paralel → 3 file Excel terdownload dalam ~3 menit

⚡ Keunggulan: Cepat karena proses bersamaan (tidak menunggu satu-satu)

💻 METODE 2: Single URL - Multi Kabupaten

Untuk Laptop Standar

� Kapan Menggunakan Metode Ini?

  • ✓ Laptop RAM terbatas (4GB atau kurang)
  • ✓ Ingin download 1 jenis laporan untuk banyak kabupaten
  • ✓ Koneksi internet standar
  • ✓ Mengutamakan stabilitas (serial processing)

� Contoh Skenario:

Download laporan Catin Bulanan untuk 23 kabupaten = Input 1 URL, pilih semua kabupaten

Ekstensi akan otomatis proses satu per satu (tidak buka banyak tab)

💻 Metode 2: Langkah-Langkah

Single URL - Multi Kabupaten

  1. 📋 Salin HANYA 1 URL laporan yang ingin didownload
  2. Paste URL ke textarea
  3. ☑️ Pilih BANYAK KABUPATEN (bisa 2-23 kabupaten)
  4. Atau gunakan tombol "Pilih Semua" untuk semua kabupaten
  5. Pilih Periode & Tahun (jika Tab Bulanan)
  6. Klik Submit
  7. 🌐 Ekstensi akan memproses satu per satu kabupaten (reuse 1 tab)
  8. 📊 Monitor progress di Tab Download

💻 Metode 2: Contoh Praktis

Download Catin Bulanan - Semua Kabupaten

� Input Textarea (1 URL):

https://siga.bkkbn.go.id/lap_catin_bul

☑️ Pilih: SEMUA 23 Kabupaten (klik "Pilih Semua")

📅 Periode: Februari | Tahun: 2025

✅ Hasil: 1 tab digunakan berulang → 23 file Excel terdownload dalam ~25 menit

💪 Keunggulan: Stabil untuk laptop RAM rendah, tidak membebani sistem

⚠️ Catatan: Lebih lambat dari Metode 1, tapi lebih aman untuk laptop standar

📊 Monitoring Progress

Tab Download - Dashboard Real-time

  • 🔵 Status: Queue (menunggu) → In Progress (proses) → Success/Failed
  • 📊 Progress Bar: Persentase keberhasilan (contoh: 18/23 = 78%)
  • ⏱️ Timestamp: Waktu mulai dan estimasi selesai
  • 📁 Last Downloaded: Info file terakhir berhasil diunduh
  • ❌ Error Log: Pesan detail jika ada kegagalan

💡 Tip: Jangan tutup browser selama proses download berlangsung

🔧 Troubleshooting

Solusi Masalah Umum

❌ Download stuck / tidak jalan
✅ Refresh halaman SIGA, pastikan sudah login, restart browser
❌ Browser freeze / hang
✅ Terlalu banyak tab terbuka (Metode 1), coba gunakan Metode 2 atau kurangi jumlah URL
❌ Banyak status "Failed"
✅ Cek URL valid, pastikan data tersedia di SIGA, cek koneksi internet
❌ File tidak muncul di Downloads
✅ Izinkan pop-up & download otomatis di browser settings

💡 Tips & Best Practices

🌐 Gunakan koneksi internet stabil minimal 2 Mbps
⏰ Download di jam kerja (server SIGA lebih responsif)
📊 Metode 1: Maksimal 10-15 URL per batch
📊 Metode 2: Bisa hingga 23 kabupaten sekaligus
🔄 Jangan tutup browser selama proses download
💾 Backup file hasil download secara berkala
🔧 Tutup tab/aplikasi lain yang tidak perlu
📁 Bersihkan folder Downloads secara rutin

❓ FAQ - Pertanyaan Umum

Q: Apakah bisa di Firefox/Safari?
A: Saat ini hanya Chrome & Edge (Chromium based)
Q: Berapa lama download 23 kabupaten?
A: Metode 1 (~15 menit), Metode 2 (~25-30 menit), tergantung koneksi
Q: Apakah data aman?
A: Ya, proses 100% lokal di browser Anda, tidak ada data dikirim ke server lain
Q: Bagaimana cara update ekstensi?
A: Download versi terbaru, hapus ekstensi lama, install yang baru
